# CLI Менеджер паролів (Варіант 13)

Консольна утиліта на мові Python, призначена для локального та безпечного збереження облікових даних (назв сервісів, логінів та паролів). Програма працює автономно у текстовому інтерфейсі командного рядка (CLI) та не потребує підключення до інтернету чи зовнішніх бібліотек.

---

## 🚀 Функціонал програми

Програма реалізує повний цикл керування записами (CRUD) через зручне інтерактивне меню:
1. **Додавання нового запису** — збереження сервісу, логіна та пароля з автоматичною валідацією полів та реєстронезалежною перевіркою на наявність дублікатів.
2. **Перегляд усіх записів** — виведення структурованої таблиці з автоматичним маскуванням паролів для забезпечення приватності.
3. **Пошук за назвою сервісу** — швидкий реєстронезалежний пошук з можливістю повного демаскування (показу) пароля за запитом користувача.
4. **Видалення запису** — безпечне видалення облікових даних із обов'язковим підтвердженням дії `(y/n)`.
5. **Безпечний вихід** — коректне завершення роботи додатка з автоматичним перехопленням примусового закриття через `Ctrl+C`.

---

## 📦 Структура проєкту

* `main.py` — основний скрипт програми, що містить бізнес-логіку та CLI-меню.
* `passwords.json` — локальна база даних у форматі JSON (створюється автоматично при першому запуску).
* `README.md` — документація та інструкція користувача.

---

## 🛠 Технічні особливості та відмовостійкість

* **Локальне зберігання:** Дані записуються у файл `passwords.json` у кодуванні UTF-8 із форматированием `indent=4`.
* **Маскування паролів:** Паролі довжиною $\le 4$ символів маскуються як `****`, а довші відображають лише перші й останні 2 символи (наприклад, `Se...23`).
* **Захист від крашів:** Обробка винятку `KeyboardInterrupt` гарантує, що при натисканні `Ctrl+C` програма безпечно збереже стан і закриється без виведення системних помилок у консоль.
